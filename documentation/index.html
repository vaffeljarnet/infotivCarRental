This project was created at an internship at Infotiv AB, Gothenburg, by Joakim Gustavsson, Johan Larsson and Maheel Dabarera. The project consists of a fake car rental booking homepage, to be used as a system under test for educational purposes.

This documentation will explain options on how to set the homepage up locally and externally, explanations of functionality and standards used over the different pages. Use the category selector to the left to jump to a specific chapter.
